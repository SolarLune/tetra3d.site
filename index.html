Tetra3D Examples

Hello!

Tetra3D is an open-source, primarily PS1-styled 3D renderer built on top of Ebitengine, written in Go, by me, SolarLune.

This is a website to showcase some examples. If you’d like to peruse the source code of this site itself, you can do so here.

Click on any of the examples in the list on the left and you’ll be able to view and play the example in the center of the page. Press F1 to show or hide help text. How fun!
